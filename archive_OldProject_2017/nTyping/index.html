If we want the search engine to read the information we intended, and we want that the information has typing effect, write code as follows.

<h1><span id='demo1'>Hello World.</span></h1>
<script>
NP.Typing.set({
    id: 'demo1',
    content: '/d12/Hello World.'
});
</script>

/d$/ is immediate delete the text. $ is the number of characters to delete.

Therefore, we can deliver the information we intend in search engine or javascript disabled.

If you replay using /l/, ignore /d$/ from the second play.

Hello World.

You can disable the cursor by writing code as follows.

<h1><span id='demo2'>I like codding!</span></h1>
<script>
NP.Typing.set({
    id: 'demo2',
    content: '/d14/I like coding!/s1500//c/'
});
</script>

/s$/ is typing pause and $ is the number of milliseconds to pause.

If you want,can be disabled the cursor using /c/.

Note: Once disabled, the cursor can not be re enabled.

I like coding!

You can write code to set the loop as follows.

<h1><span id='demo3'>I can be typing infinitely!</span></h1>
<script>
NP.Typing.set({
    id: 'demo3',
    content: '/d11/infinitely!/l/'
});
</script>

Ignore /d$/ from the second play.

I can be typing infinitely!

If you want to write /, write the following code.

<h1><span id='demo4'></span></h1>
<script>
NP.Typing.set({
    id: 'demo4',
    content: 'I can write // and << !!/s1000//l/'
});
</script>

If you want to use tag, write the following code.

<h1><span id='demo5'></span></h1>
<script>
NP.Typing.set({
    id: 'demo5',
    content: 'I can to use <s>tabg</s>.<br>Its/b7/g</b>.<br>Its perfect!/l/'
});
</script>

If backspace from outside to inside the tag, you must to re-declare the closing tag of that tag.

Unclosed tag type is recognized as a one character.

If you want to set the replacement text, write like as follows code.

<h1><span id='demo6'></span></h1>
<script>
NP.Typing.set({
    id: 'demo6',
    content: 'Check with screen reader./l/',
    altText: 'I love coding.'
});
</script>

A custom function can be used by writing like the following code.

<h1><span id='demo7'></span></h1>
<script>
NP.Typing.set({
    id: 'demo7',
    content: 'Cus/e/tom/e/ function./e//l/',
    typeSpeed: 500,
    event: myFunc1
});
function myFunc1(obj) {
    console.log('Run!');
    console.log(obj);
}
</script>
